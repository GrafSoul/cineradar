# 🎬 CineRadar

Ежеквартальные дайджесты лучших фильмов и сериалов.

🌐 **Сайт:** [grafsoul.github.io/cineradar](https://grafsoul.github.io/cineradar/)

## Что внутри

Стильные HTML-страницы с карточками фильмов и сериалов — рейтинги (IMDb, Rotten Tomatoes, Кинопоиск), платформы, редакторские описания с характером и ссылки на русские тизеры в YouTube. Интерактивные фильтры по жанрам, анимации, тёмная кинематографичная тема. Без фреймворков — чистый HTML/CSS/JS.

## Структура

```
cineradar/
├── index.html                  # Главная — хаб всех дайджестов
├── movies-digest-2026.html     # Кино Q1 2026
├── tv-series-digest-2026.html  # Сериалы Q1 2026
└── README.md
```

## Возможности

- 🌙 Тёмная кинематографичная тема
- 🏷️ Интерактивные фильтры по жанрам и категориям
- ▶️ Ссылки на русские тизеры в YouTube
- ⭐ Рейтинги IMDb, Rotten Tomatoes, Кинопоиск
- 📱 Адаптивная вёрстка
- 🔄 Обновляется каждые 3 месяца

## Деплой

Работает на GitHub Pages. Просто включите Pages в настройках репозитория:

**Settings → Pages → Source: Deploy from a branch → Branch: main, / (root) → Save**
