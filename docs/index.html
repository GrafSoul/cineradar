Как это работает

Каждые три месяца здесь появляется новая пара дайджестов — кино и сериалы. Внутри — карточки с описаниями, рейтингами, платформами и ссылками на русские тизеры в YouTube. Фильтры помогают быстро найти нужный жанр. Пустые карточки — это слоты для будущих выпусков. Проект живёт на GitHub Pages.
